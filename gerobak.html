Gerobak / Paket Kemitraan

Bagian ini menjelaskan cara menambahkan dan mengelola paket kemitraan dalam bentuk gerobak melalui CMS Sikriuk.

Menambah Gerobak / Paket Kemitraan

Ikuti langkah-langkah berikut untuk menambahkan paket kemitraan:

  1. Buka menu Tambah Gerobak di admin.
  2. Isi data yang diperlukan termasuk gambar (ukuran gambar: 418x547).
  3. Klik tombol Next.
  4. Tambahkan benefit yang disediakan oleh paket kemitraan.
  5. Klik Simpan untuk menyelesaikan proses.

Mengedit & Menghapus Paket Kemitraan

Untuk mengelola paket kemitraan yang sudah ada:

  1. Buka daftar paket kemitraan.
  2. Klik tombol Edit pada item untuk memperbarui data.
  3. Klik tombol Hapus pada item untuk menghapus paket kemitraan.

Tampilan

Paket kemitraan akan ditampilkan di halaman kemitraan pada website.
