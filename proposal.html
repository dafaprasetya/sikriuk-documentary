Proposal Kemitraan

Menu Proposal Kemitraan berfungsi untuk mengunggah dan mengelola file proposal resmi yang dapat ditampilkan di halaman website publik. File ini akan menjadi bahan pertimbangan bagi calon mitra untuk memahami konsep, skema, dan keuntungan bergabung dalam kemitraan.

Menambahkan Proposal Kemitraan

Untuk mengunggah file proposal kemitraan, ikuti langkah-langkah berikut:

  1. Pada sidebar, klik menu Proposal di bawah bagian Kemitraan.
  2. Masuk ke tab Upload Proposal.
  3. Klik tombol Choose File dan pilih file dengan format .pdf.
  4. Pastikan file sudah benar, lalu klik tombol Save Change.

Tampilan Proposal

File proposal yang berhasil diunggah akan muncul sebagai link di halaman publik (frontend). Pengunjung atau calon mitra dapat mengakses dan mengunduh file tersebut untuk dipelajari lebih lanjut.
