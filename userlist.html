Manajemen User

Menu User digunakan untuk menambah dan mengelola pengguna sistem. Fitur ini hanya dapat diakses oleh user dengan role Admin. Role selain admin tidak memiliki hak akses untuk melihat atau menambahkan user.

Menambah User Baru

Untuk menambahkan user baru ke dalam sistem, ikuti langkah-langkah berikut:

  1. Buka tab User di sidebar.
  2. Pilih tab Tambah User.
  3. Isi field Name dengan nama pengguna baru.
  4. Masukkan Email Address aktif.
  5. Isi Password dan konfirmasi di Confirm Password.
  6. Klik tombol Register untuk menyimpan data.

Catatan

  • User yang ditambahkan akan memiliki hak akses sesuai role yang ditetapkan secara default atau melalui pengaturan tambahan.
  • Hanya Admin yang dapat mengakses dan memodifikasi daftar user.
